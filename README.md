Overview
===
Library for interacting with the SmugMug 1.3.0 API using OAuth authentication.

Compability
-
Tested against Ruby 1.8.7, 1.9.2, 2.0.0, RBX and JRuby, build history is available [here](http://travis-ci.org/zanker/ruby-smugmug).
